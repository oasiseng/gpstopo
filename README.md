# GPSTopo Codex

**GPSTopo Technologies** is building a modern mapping and site-data platform for builders, designers, and homeowners.  
This Codex documents our internal quality-control standards, workflows, and open best practices for the mapping industry.

> ⚠️ Disclaimer  
> This repository is for educational purposes and general operational transparency.  
> GPSTopo does not provide legal land-surveying services; licensed survey professionals perform field verification where required by law.

## Repository Overview

This repository captures the processes, templates, and tools that drive GPSTopo's operations.  
It is designed to be transparent, collaborative, and extensible for teams building dependable geospatial products.

- **docs/** — Detailed guides on quality control, data processing, field operations, and client delivery.  
- **templates/** — Reusable checklists and forms for day-to-day operations.  
- **flowcharts/** — Visual process references for onboarding and training.  
- **systems/** — Descriptions of automation, integrations, and supporting infrastructure.  
- **changelog/** — Release notes and historical context for major documentation updates.
