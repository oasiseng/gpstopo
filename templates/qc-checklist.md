# QC Checklist

Use this checklist during draft and final reviews. Mark each item as complete or
note outstanding issues with references to supporting documents.

- [ ] Project metadata verified (name, location, CRS, datum)
- [ ] Raw data archived in staging with checksums
- [ ] Processing log reviewed and parameters documented
- [ ] Geometry accuracy validated against control points
- [ ] Contours and annotations match project specifications
- [ ] Deliverable filenames follow GPSTopo conventions
- [ ] Disclaimers and revision table present on all PDF outputs
- [ ] Companion metadata summary prepared
- [ ] Final package archived to `/Finals/YYYY/ProjectName/`
- [ ] Client notification drafted and reviewed
