# Glossary

Definitions for key terms used throughout GPSTopo documentation.

- **Control Point**: A surveyed location with known coordinates used to validate
  positioning accuracy.
- **DEM (Digital Elevation Model)**: A raster representation of terrain heights.
- **Drone Mission Log**: Record of flight parameters, observations, and
  anomalies for each unmanned aerial system deployment.
- **Metadata Snapshot**: Summary of collection details (equipment, date, datum)
  archived with each deliverable version.
- **RTK (Real-Time Kinematic)**: GPS correction technique that improves
  positional accuracy using a base and rover setup.
- **Staging Area**: Secure storage location where raw datasets are uploaded
  prior to processing or review.
- **Version Tag**: Identifier following the `vYYYY.MM.DD-Initials` convention to
  track revisions of a dataset or document.
